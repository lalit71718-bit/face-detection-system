# Face Detection System

This is a simple Face Detection System built using Python, OpenCV, and Streamlit.

## Features

- Upload an image
- Detect human faces
- Draw rectangles around detected faces
- Show total number of faces detected
- Simple web interface using Streamlit

## Technologies Used

- Python
- OpenCV
- Streamlit
- NumPy
